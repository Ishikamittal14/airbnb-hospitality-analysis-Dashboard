# 🏠 Airbnb Hospitality Dashboard (Power BI)

## 📊 Project Overview

This project is an interactive **Airbnb Hospitality Dashboard** built using Power BI.
It provides insights into listings, pricing trends, host activity, and geographical distribution.

The dashboard helps in understanding:

* Distribution of listings by room type
* Pricing patterns across different property types
* Host behavior and growth over time
* Cancellation policies followed by hosts
* Location-based insights using map visualization

## 🎯 Objectives

* Analyze Airbnb listing data
* Identify trends in pricing and revenue
* Understand host participation and response patterns
* Visualize geographical distribution of listings
* Build an interactive and user-friendly dashboard

## 📌 Key Features

* 📍 **Map Visualization** – Shows global distribution of listings
* 📈 **Line Chart** – Trends of listings over time
* 📊 **Bar & Column Charts** – Comparison across cities and room types
* 🍩 **Donut Chart** – Room type distribution
* 🥧 **Pie Chart** – Cancellation policy distribution
* 🔵 **Scatter Plot** – Relationship between price and revenue
* 🎛️ **Slicers** – Interactive filtering by City and Room Type

## 📂 Dataset Details

The dataset contains **55,000+ records** with the following fields:

* Listing ID
* Host details
* City & Country
* Room Type
* Price per night
* Reviews & Ratings
* Cancellation Policy
* Availability & Revenue

## 🛠️ Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization
